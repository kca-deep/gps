📚 REST API 문서

POST/api/chat/session

설명: 새로운 채팅 세션을 생성합니다.

{ "location": { "latitude": 37.5665, "longitude": 126.9780 } }

POST/api/chat/message

설명: 채팅 메시지를 처리하고 AI 응답을 반환합니다.

{ "session_id": "session_123", "message": "서울 무선국 검색", "location": { "latitude": 37.5665, "longitude": 126.9780 } }

GET/api/search/stations

설명: 무선국을 검색합니다.

파라미터: q (검색어), page (페이지), per_page (페이지당 결과 수)

GET /api/search/stations?q=서울&page=1&per_page=10

GET/api/search/chosung

설명: 초성으로 무선국을 검색합니다.

GET /api/search/chosung?q=ㅅㅇ&limit=10

GET/api/stations

설명: 무선국 목록을 조회합니다.

GET /api/stations?page=1&per_page=10&status=active

POST/api/stations

설명: 새로운 무선국을 등록합니다.

{ "name": "서울기지국_001", "latitude": 37.5665, "longitude": 126.9780, "frequency": "2.4GHz", "power": "10W", "station_type": "기지국" }

POST/api/stations/check-duplicate

설명: 위치 기반 중복 무선국을 확인합니다.

{ "latitude": 37.5665, "longitude": 126.9780, "name": "서울기지국_001", "radius": 100 }

GET/health

설명: 시스템 전체 상태를 확인합니다.

{ "status": "healthy", "components": { "database": "connected", "openai": "configured", "cache": "active" } }

💡 사용 예시

JavaScript로 API 호출하기

// 채팅 세션 생성 const response = await fetch('/api/chat/session', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ location: { latitude: 37.5665, longitude: 126.9780 } }) }); const session = await response.json(); console.log('세션 ID:', session.session_id); // 메시지 전송 const messageResponse = await fetch('/api/chat/message', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ session_id: session.session_id, message: '서울 무선국 검색해줘' }) }); const result = await messageResponse.json(); console.log('AI 응답:', result.response);
